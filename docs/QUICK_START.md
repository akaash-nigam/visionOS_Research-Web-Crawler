### Step 2: Add Your First Source (1 minute)

**Option A: Quick Web Scrape** (easiest)

1. Tap **"Add Source"**
2. Select **"From URL"**
3. Paste a URL (try: `https://en.wikipedia.org/wiki/Artificial_intelligence`)
4. Tap **"Scrape"**
5. Review auto-filled information
6. Tap **"Save"**

**Option B: Manual Entry** (more control)

1. Tap **"Add Source"**
2. Select **"Add Manually"**
3. Fill in at minimum:
   - Title: "Introduction to AI"
   - Authors: "John Doe"
   - Type: "Article"
4. Tap **"Save"**

âœ… **You've added your first source!**

---

### Step 3: View in 3D (30 seconds)

1. Tap **"Graph View"** button
2. Your source appears as a colored sphere
3. Use natural gestures:
   - **Drag** to rotate the view
   - **Pinch** to zoom in/out
   - **Tap** a node to select it

âœ… **You're experiencing spatial computing for research!**

---

### Step 4: Add Another Source & Connect Them (1 minute)

1. Return to source list (back button)
2. Add a second source (any method)
3. In the graph, **tap and hold** the first source
4. **Drag toward** the second source
5. Release when hovering over it
6. Select relationship type: **"References"**
7. Tap **"Create Connection"**

âœ… **You've created your first knowledge graph!**

---

### Step 5: Generate a Citation (30 seconds)

1. Tap any source node (or select from list)
2. Tap **"Copy Citation"**
3. Choose format: **APA**, **MLA**, or **Chicago**
4. Citation copied to clipboard!
5. Paste anywhere (Messages, Notes, Pages, etc.)

âœ… **Perfect citations in seconds!**
