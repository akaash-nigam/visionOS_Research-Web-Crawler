# Research Web Crawler

> 3D knowledge graph for research and information discovery on Apple Vision Pro

## Overview

Research Web Crawler transforms research by visualizing information sources as an interactive 3D node network, enabling users to drag connections between concepts, receive AI suggestions for missing links, and auto-organize citations spatially.

## Key Features

- **3D Knowledge Graph** - Research sources as interconnected nodes in space
- **Drag-to-Connect** - Create relationships by literally connecting nodes
- **AI Missing Link Suggestions** - AI identifies gaps and recommends sources
- **Smart Citation Management** - Auto-formatting in APA, MLA, Chicago, etc.
- **Multi-Modal Content Capture** - Add sources via web, PDF, voice, photos
- **Collaborative Research** - Shared knowledge graphs with team members
